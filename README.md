# Problema del Viajante (TSP) - Algoritmos Genéticos

Resolución del Problema del Viajante de Comercio (TSP - Traveling Salesperson Problem) mediante la implementación de un Algoritmo Genético (AG). 

## Características y Operadores

- **Selección de Padres**: Ruleta, Torneo.
- **Cruce**: Cruce de Arcos, Cruce de Orden .
- **Mutación**: Intercambio, Inversión.
- **Selección de Padres**: Ruleta, Torneo
- **Selección de Sobrevivientes**: Ruleta, Steady State, Hijos por Padres (Generacional).

Para la interfaz gráfica se utilizó la librería [FlatLaf](https://www.formdev.com/flatlaf/) para mejorar el aspecto visual de los componentes de Swing de Java.

## Cómo ejecutarlo

1. Clonar el repositorio.
2. Abrir la carpeta en el IDE (IntelliJ, Eclipse o Vscode).
3. Asegurarse de agregar el archivo `flatlaf-3.7.jar` (ubicado en la carpeta `Librerias`)
4. Ejecutar la clase `Main.java`.
